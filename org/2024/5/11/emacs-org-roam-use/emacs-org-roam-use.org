** 最佳实践

qnap 使用 webdav

https://post.smzdm.com/p/a4pmggzw/

什么是最佳实践呢？就是用 qnap 做一个 webdav 可以挂载到我的工作机上，那样的话，我就会有
一套永远不断发展的个人科研库了。时间长了之后，我可能学到更多的东西。最后。最终的目的还
是学习东西，学习有用的知识。

小米金色飞贼的科研流程总结的非常好，其中也提到了好几个人名。没想到 orgmode 是 2003 年才
出现的软件，原来这个世界的历史并不是很长。

通过 zotero 导出 sqlite 的数据库。基实本质上无需 zotero ，只需要 org-roam-bibtex 即可，
可能是 zotero 这个软件 UI 交互更好一些吧。
