* NetworkManager 配置文件

#+begin_src
TYPE=Ethernet
PROXY_METHOD=none
BROWSER_ONLY=no
BOOTPROTO=none
IPADDR=10.10.xx.xx
PREFIX=24
GATEWAY=10.10.xx.1
DEFROUTE=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_FAILURE_FATAL=no
IPV6_ADDR_GEN_MODE=stable-privacy
NAME="有线连接 1"
UUID=88b5a874-9d58-4da6-9923-248e1470ad39
DEVICE=enp5s0f2
ONBOOT=yes
DNS1=10.10.0.xx
DNS2=10.10.0.xx
PEERDNS=no
#+end_src




保证重启网络正常，比较重要的两点是：DEVICE 这个是绑定设备的，另一个是 ONBOOT=yes ，这两个配对了，网络就没有
问题了。
