* gitlab 个人令牌

https://gitlab.com/-/profile/personal_access_tokens

建一个新的 personal access token

#+begin_src bash
 git remote -v
 git remote  remove origin
 git remote add origin https://oauth2:"personal-key"@gitlab.com/Allinaent/"repository"
 git status
 git pull
 git branch --set-upstream-to=origin/main main
#+end_src
